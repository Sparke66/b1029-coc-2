Porting Static Website to Dynamic

Simply change the extension of each webpage from .html to .php (make sure to update any links or references accordingly).

PHP Partials

PHP partials allow you to reuse code snippets across multiple pages. Create a separate PHP file for the header, footer, and any other common elements, then include them in your main pages using the <?php include 'header.php'; ?> syntax.

header.php

<header>
    <h1 id="title"><?php echo $title; ?> - INFO 2300</h1>

    <nav id="menu">
      <ul>
        <li><a href="index.php">Home</a></li>
        <li><a href="citations.php">Citations</a></li>
      </ul>
    </nav>
</header>

webpage_name.php

<body>
    ...
    <?php include('includes/header.php'); ?>
    ...
    <main>
    ...
